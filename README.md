# quokka-xtools

# Installation

```
-- download
$ git clone https://github.com/shellever/quokka-xtools.git
-- configure
$ cd quokka-xtools
$ source xsetup.sh
```

if you modify the file `config/xtools_config.sh`, and want to take an effect right now,
please type the following:

```
$ source ~/.bashrc
```
